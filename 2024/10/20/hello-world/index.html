Library Genesis 简介

Library Genesis (简称 LibGen) 是一个非常优秀的外文图书资料文献下载工具。LibGen 又名创世纪图书馆，是一个电子书下载平台，提供了大量优质的电子书，尤其是英文课本图书，全网免费下载！

Library Genesis 号称是帮助全人类实现知识无版权传播的计划。该网站（https://libgen.is/）上有很多学术论文，下载方便，还有很多外文书籍和中文书籍，几乎每天都在更新。

这是一个神奇的网站，基本上所有的外文书籍和论文都可以搜索并下载，最新的论文也可以找到。该平台拥有超过 80 万本图书供用户免费下载，其中 99% 是科学著作，且许多是市面上难以购买到的书籍。

搜索 Library Genesis 打开主页，其首页界面如下图所示：

如上图所示，默认的检索范围是选择 LibGen(Sci-Tec) ，也就是默认的选择是检索电子书。如果大家要检索文章，就请选择 Scientific Articles。如果大家要查询期刊，就选择 Magazines。既可以输入文献名字或关键词，也可以输入 DOI 进行检索。

当在默认的检索范围检索电子书时，检索框中输入的是书的名字。当要检索的是文章时，检索框中输入的是文章的 DOI 编码或 PMID 。当检索的是期刊时，则输入的是期刊名字。

假设我们要检索一篇文章，该文章的 DOI 编号是 10.1080/10408398.2017.1362630。我们将此 DOI 号在检索框中输入并检索（注意，选择的检索范围是 Scientific）。

访问工具

1. 介绍

支持 Shadowsocks、VMess、VLess 等多种协议，提供安全高效的网络连接。

2. 安装步骤

Windows / Android：

  1. 点击这里下载 密码：d1ko
  2. 安装过程中，若遇到安全警告，请点击“显示详细信息”，并选择“仍要运行”。
  3. 安装完成后，按照以下步骤进行配置。

3. 添加配置

  1. 打开 软件，点击左上角的设置按钮。
  2. 选择 添加配置。
  3. 将 Clash/V2Ray/SS 的配置链接粘贴到输入框中，或通过二维码扫描添加。以下是免费截点网站。
    Open Proxy List

选择添加配置，全选复制：V2Ray Base64 配置文件

  1. 点击 添加 按钮，返回主界面。

4. 连接服务器

  1. 软件默认会为你选择一个服务器。你也可以点击服务器名称重新选择。
  2. 点击 连接 按钮开始使用。

优化连接：

  • 点击主界面的 选择服务器，并选择 延迟检测 查看各服务器的延迟时间。延迟值越低，速度越快。
  • 可以使用 自动选择 功能来挑选最佳服务器。

5. 进阶功能

  • 自定义路由规则：根据需要设置自定义规则，以更好地管理流量。
  • 自动选择服务器：根据性能数据自动选择最佳服务器。

6. 常见问题解决

  • 连接问题：启用 FakeIP DNS 功能，防止因 DNS 污染导致的连接问题。
